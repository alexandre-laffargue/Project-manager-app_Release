## 1. Introduction

L’application **Project Manager App** est un outil web de gestion de projet inspiré de **Jira** et **Trello**.  
Elle permet de gérer des **tâches**, **issues**, **sprints** et **releases** à travers trois vues principales :

- **Backlog** → gestion et priorisation des issues  
- **Tableau Kanban** → suivi visuel des tâches  
- **Chronologie (vue Gantt)** → planification temporelle des sprints et issues

## 2. Technologies utilisées

| Domaine | Technologie | Description |
|----------|--------------|-------------|
| **Frontend** | Vue.js 3 + Vite | Framework réactif pour l’interface utilisateur |
| **Backend** | Node.js + Express | API REST performante |
| **Base de données** | MongoDB (Mongoose) | Stockage des tâches, sprints et utilisateurs |
| **Authentification** | JWT (JSON Web Token) | Sécurisation des accès |
| **Tests** | Vitest + Selenium | Tests unitaires et end-to-end |
| **CI/CD** | GitHub Actions | Automatisation du build et des tests |
| **Outils** | Docker, ESLint, Prettier, Husky | Standardisation du code et environnement maîtrisé |
